# ForensicFace Developer Documentation

This folder contains developer-facing documentation for understanding and
evolving the `forensicface` codebase.

- [Architecture and contributor guide](architecture.md): functional blocks,
  end-user workflows, data flow, and where to start when contributing.
- [Simplification status](simplification-plan.md): what was extracted from the
  original `ForensicFace` class, what the current module boundaries are, and
  which API decisions remain intentionally deferred.

Generated documentation artifacts, such as HTML exports, should be regenerated
intentionally and reviewed separately from source markdown edits.
